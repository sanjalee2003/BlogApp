# BlogApp
Blog Application built using HTML, CSS, and JavaScript for the frontend, PHP for the backend, and MySQL as the database

Functionalities

1. User Authentication & Authorization
Users can register, log in, and log out.
Only authenticated users can create, update, or delete their own blogs.
Authorization ensures that a user cannot edit or delete blogs created by other users.

3. Blog Management
Users are able to:

Add new blogs.
Read blogs — all blogs are displayed on the home page.
Update their own blogs.
Delete their own blogs.

5. Frontend Features
Home Page: Displays a list of all blogs.
Single Blog View Page: Shows blog title, content, author, date created, and date updated.
Blog Editor Pages: Used for creating and updating blogs.
Responsive & Clean UI: Built with HTML, CSS, and JavaScript.

7. Database
The application uses a database with two main tables:
User table: Stores user details for registration and login.
blogPost table: Stores blog details such as title, content, author, and created/updated dates.
